                    ⚠️ TROUBLESHOOTING GUIDE

═══════════════════════════════════════════════════════════════════════

ISSUE 1: "ModuleNotFoundError"
───────────────────────────────
Problem: Missing package in requirements.txt
Solution: Add missing package to requirements.txt and push:
    echo "missing-package" >> requirements.txt
    git add requirements.txt
    git commit -m "Add missing dependency"
    git push


ISSUE 2: "File not found: *.pkl"
─────────────────────────────────
Problem: Model files not uploaded to GitHub
Solution: Check .gitignore doesn't exclude .pkl files
    # Remove *.pkl from .gitignore if present
    git add *.pkl
    git commit -m "Add model files"
    git push


ISSUE 3: App Crashes on Startup
────────────────────────────────
Problem: Error in app.py
Solution: Check Streamlit Cloud logs:
    1. Go to Streamlit Cloud dashboard
    2. Click on your app
    3. View logs in bottom panel
    4. Fix error locally
    5. Push changes


ISSUE 4: Large File Warning (>100MB)
─────────────────────────────────────
Problem: random_forest_model.pkl is ~200MB
Solution: Use Git LFS (Large File Storage)
    git lfs install
    git lfs track "*.pkl"
    git add .gitattributes
    git commit -m "Track large files with LFS"
    git push


ISSUE 5: Deployment Timeout
────────────────────────────
Problem: Deployment takes too long
Solution: 
    1. Check file sizes (large models slow deployment)
    2. Restart deployment from Streamlit dashboard
    3. Contact Streamlit support if persists
